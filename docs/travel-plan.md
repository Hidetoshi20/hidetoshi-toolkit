# 广东国庆自驾游（深圳 + 万绿湖）

**行程区间**：2025-10-04（深圳城市日）→ 2025-10-06（万绿湖返程）

**参与人员**：深圳 2 人 + 广州 2 人（10/04 上午深圳会合）  
**交通工具**：纯电 SUV（续航约 400–500 km）  
**随行宠物**：小狗随行日程为 10/04~10/06；避免公共交通，优先自驾

## 变更重点（相较旧版）
- 已取消长隆行程，全部改为深圳城市日 + 万绿湖两日
- 广州两人 10/04 上午抵达深圳后合流，当晚住深圳，10/05 早上四人一车自驾前往万绿湖
- 深圳玩法聚焦携宠海边遛狗与海堤步道，可参考旧版在地经验但需现场确认宠物政策

## TL;DR
- **10/04 深圳**：广州 2 人上午抵达 → 会合后自驾探索海堤/礁石步道（不一定下沙）→ 夜宿深圳宠物友好民宿/酒店并补能。
- **10/05 万绿湖 D1**：06:30 左右自驾出发（约 180 km）→ 11:00 前入住湖畔可宠民宿 → 下午水月湾步道或桨板体验 → 日落晚餐。
- **10/06 万绿湖 D2**：06:15 左右日出 → 上午二段活动（桨板/游船/徒步）→ 午餐后整理返程 → 下午途经河源/广州补能送人 → 晚间抵深圳。
- **出行提醒**：深圳地铁及多数公共交通仅放行服务犬，携宠行程以自驾为主；湖区活动关注风力与雷雨预警。

## 行程脉络
### 10/04（周六）深圳城市日
- **08:30 前** 广州 2 人抵深圳（优先自驾；如高铁需提前确认宠物托运政策）。
- **10:00 左右** 四人会合后检查装备/狗狗状态，确认宠物友好酒店入住手续。
- **白天路线（任选 1–2 条，可串联）**  
  - `大鹏半岛非核心海段`：西冲/东冲周边礁石区或海堤，避开管控严的主沙滩；全程牵引、按现场要求执行。  
  - `城市海堤步道`：深圳湾公园海堤、后海石板路等开阔步道，适合散步拍照、补给方便。  
  - `备用室内/轻户外`：华侨城创意园、宝安滨海廊桥等开放空间，遇雨时作为备选。
- **晚间**：入住深圳宠物友好民宿/酒店；顺路快充补到 70–80%；打包第二日早餐/路餐。

### 10/05（周日）万绿湖 D1
- **06:30** 深圳出发 → 经深汕高速/广惠高速，途中服务区短停；若需补能，优先河源市区/新港镇快充站。
- **10:30 前** 抵达万绿湖 → 办理湖畔民宿入住（确认宠物押金/清洁费）。
- **午餐**：自带简餐或民宿供应。
- **下午活动二选一**  
  - `A`：水月湾景区步道 + 文化长廊 + 亲水平台拍照。  
  - `B`：背风安全下水点桨板/皮划艇 1–2 小时（救生衣、脚绳、涉水鞋齐备）。  
- **傍晚**：湖畔看日落，晚餐按“民宿预订”或“回市区用餐”预案执行；晚间整理设备并规划次日活动。

### 10/06（周一）万绿湖 D2
- **06:00** 起床，06:15 左右日出观察点卡位（提前 20–30 分钟到场）。
- **早餐**：民宿或自带轻食。
- **上午活动（择一或串联）**：再上桨板、乘游船登岛、新丰江大坝外观、轻徒步摄影、镜花缘等。
- **午餐**：民宿加餐或回河源市区补给。
- **下午**：收拾行李 → 河源/景区入口快充至 70–80% → 返程（途经广州送 2 人）→ 晚间回到深圳。若想放松，可延长至 10/07 早上返程。

## 行前检查清单
- 住宿：深圳 10/04 夜、万绿湖 10/05 夜均确认可携宠、费用及是否提供早餐/晚餐。
- 宠物：疫苗/健康证明、牵引、便携碗、垫子、防虫药、宠物救生衣；提前适应长途车程。
- 交通与补能：深圳出城、河源市区、景区入口各准备“两主两备”充电站（含功率/收费/停车策略）。
- 水上项目：桨板来源（深圳自带/河源自提/景区租赁）、救生衣规格、脚绳/涉水鞋是否齐备。
- 装备：手机防水套 2 个/人 +1 备用、防水干袋、密封袋、车载冰箱或保冷箱。
- 餐食：湖畔餐饮有限，准备简餐、饮用水 ≥10 L、咖啡/热水；列好河源市区备选餐馆。
- 下水点：提前在地图标注 3–5 个背风、坡度缓、有停车位的下水口，遇风浪或人多及时换点。
- 天气：出发前 72/48/24 小时滚动查看降雨、风力、雷电预警，必要时调整或取消水上项目。

## 充电与里程提示
- 深圳⇄万绿湖往返约 360–400 km，满载+空调用电下预估消耗 60–75% 电量。
- 去程建议：深圳出发前补满 → 河源市区/景区口补能一次 → 入住后保持 60% 以上。
- 返程建议：10/06 中午或下午离开前再补至 70–80%，途经广州送人后仍保留 ≥20% 余量。
- 建议准备不同运营商的充电 App，并关注节假日排队情况。

## 装备与物资
- 证件支付：身份证、驾照、现金/移动支付。
- 防晒防护：防晒霜、遮阳帽、墨镜、防蚊喷雾、一次性雨披。
- 医药：创可贴、消炎止痛、晕车药、肠胃药、常用处方、宠物常备药。
- 服装：快干衣裤、轻羽绒/薄外套、涉水鞋或溯溪鞋、替换毛巾。
- 水上：救生衣（每人一件）、脚绳、桨板打气/修补工具、手机防水套、防水干袋。
- 饮食：简餐、零食、水果、饮用水 ≥10 L、保冷冰袋、餐具、湿巾、垃圾袋。
- 车载：充电线/转换头、车载冰箱或保温箱、打气泵/应急启动电源（如有）。

## 风险与应急
- 电量 <30% 立即就近快充，保持至少 2 个替补站点。
- 遇雷雨、大风或水体能见度差，停止下水并转为步道/观景/市区游。
- 宠物出现中暑或肠胃问题及时就医（提前保存河源、深圳宠物医院联系电话）。
- 交通拥堵时可利用国省道绕行或错峰出发；必要时调整返程日期。
- 夜间驾驶保持两人轮换，提高休息质量。
