# Hidetoshi 的脚本与配置

本仓库是我的个人收藏，用于存放常用的软件配置、实用脚本和一些杂项文档。它作为一个集中的、受版本控制的空间，用于管理我的个人数字工作环境。

## 仓库内容

在这里，你会发现以下内容：

*   **软件配置:** 我经常使用的工具的设置（例如 `clash.yaml`）。
*   **开发规范:** 我遵守的个人或专业准则（例如 `frontend-development-standards`）。
*   **实用脚本:** 用于自动化个人任务的小脚本。
*   **杂项文档:** 其他不适合放入更结构化的知识库的文档。

## 范围

本仓库专门用于存放配置和独立文档。它**不**用于存储知识类内容、学习笔记或我的“第二大脑”。所有以知识为核心的笔记都在我的知识仓库（Obsidian）中单独管理。

## 使用方式

所有内容都通过目录进行组织，并使用 Git 进行管理，旨在保持整洁和易于访问。
