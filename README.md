# Hidetoshi 的脚本与配置

个人脚本、配置与单体文档的集中仓库，用于复用、迁移和备份工作环境。

## 内容

- 软件配置（如 `clash.yaml`）
- 开发规范与协作流程
- 实用脚本与工具
- 杂项文档（非知识库内容）

> 知识类内容放在 Obsidian，不在此仓库维护。

## 快速开始

- 克隆后按需取用配置与脚本
- 重要修改建议基于分支提交，保持可回溯
- 敏感信息放入环境变量或外部私有文件

## 目录速览

- `dotfiles/`：个人电脑配置（Chezmoi source）
- `docs/`：方法论与辅助文档
- `DEVELOPMENT.md`：使用与开发规范
