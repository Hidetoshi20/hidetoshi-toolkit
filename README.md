# Hidetoshi 的脚本与配置

集中管理个人常用“脚本、配置与单体文档”的版本库，用于快速复用、迁移与备份个人数字工作环境。

## 内容结构

- 软件配置：常用工具的配置文件（例如 `clash.yaml`）
- 开发规范：个人/团队使用的工程与协作规范（如前端规范等）
- 实用脚本：用于自动化日常任务的脚本集合
- 杂项文档：不适合进入知识库的独立说明文档

> 注：知识类内容与系统化笔记放在独立的知识库（Obsidian）中，不在此仓库维护。

## 快速开始

- 克隆后按需取用配置与脚本
- 重要修改建议基于分支提交，保持可回溯
- 敏感信息（令牌/私钥等）放入环境变量或外部私有文件，避免直接写入仓库

> 📖 **使用指南**：查看 [开发规范与操作指南](DEVELOPMENT.md) 了解配置文件使用、脚本操作和开发流程。

## 目录结构

## 仓库约定

- **命名规范**：文件与目录使用 `kebab-case`
- **文档原则**：配置文件力求自解释，复杂功能提供独立文档
- **安全要求**：敏感信息使用环境变量，避免直接提交到仓库
