# Web 开发规范

本目录包含通用的 Web 开发相关规范，这些规范不特定于任何具体的语言或框架，旨在为 Web 项目提供一致的开发标准。

- **[项目结构规范](./project-structure.md)** - 关于如何组织 Web 项目的目录和文件。
- **[测试开发规范](./testing-standards.md)** - 关于 Web 项目的测试策略、工具和最佳实践。
- **[React 开发规范](../development-framework/react/README.md)** - React 框架开发规范。
- **[TypeScript 开发规范](../language/ts/README.md)** - TypeScript 语言开发规范。
- **[Less 开发规范](../language/less/README.md)** - Less 语言开发规范。
